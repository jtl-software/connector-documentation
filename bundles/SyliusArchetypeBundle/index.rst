SyliusArchetypeBundle
=====================

This bundle provides easy integration of the :doc:`Sylius Archetype component </components/Archetype/index>`
with any Symfony full-stack application.

Sylius uses this bundle internally for its product catalog to manage and quickly create products with a predefined set
of attributes and options.
